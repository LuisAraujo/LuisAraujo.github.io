Como instalar

A biblioteca MediaComp usa a biblioteca Pillow para as funções de imagem, assim é preciso instalar a biblioteca Pillow.

Passos:
  • Instalar a Pillow;li>
  • Copie a pasta mediacomp para a sua pasta de bibliotecas python;
  • Importe a biblioteca MediaComp para o seu arquivo (veja nos exemplos).
