Instalando JSDoc
Projetos em javascritp tem crescido consideravelmente. Hoje o js é a linguagem mais usada nos projetos do github. Com isso, se torna cada vez mais preciso documentar o seu código, o JSDoc é uma API simples e de fácil uso que permite construir sua documentação através dos comentários do seu código. Nesse post você aprederá como instalar e usar a API JSDoc para documentar o seu projeto!
