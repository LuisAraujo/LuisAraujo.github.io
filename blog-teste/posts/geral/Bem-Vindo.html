Arquitetura do blog

Como vocês deve saber, no git não há suporte para linguagem sever-side, o que significar que essa aplicação não usa nenhuma linguagem como php, python e etc... É puramente HTML, Javascript e CSS. Muita gente usa esse tipo de site, pois há alguns "criadores" de sites estáticos como o Jekyll.

Muitas pessoas usam esses "criadores" e se sentem confortáveis com eles, mas eu não! Desse modo, esse site é puramente estático sem uso de nehuma dessas ferramentas. No entanto, mais para frente, penso em criar um simples gerador com PHP. Por equanto eu vou criando file-to-file mesmo.
