Ementa da disciplina

Explora o conhecimento de soluções clássicas de problemas por meio de abstração utilizando conjuntos de dados, operações e representações de listas, pilhas, filas e árvores, apoiados em métodos e técnicas, tendo como subsídio uma linguagem de programação. Apresenta os principais conceitos do paradigma de orientação a objetos, com ênfase em suas principais características e recursos oferecidos. São examinadas implementações de aplicações práticas, baseadas em uma linguagem de programação orientada a objetos e um ambiente integrado de desenvolvimento. Examina o projeto e implementação de sistemas computacionais onipresentes, ou seja, aplicações móveis, acessíveis por meio de quaisquer dispositivos computacionais e integráveis com aplicações existentes. Além disso, enfatiza aspectos práticos, por meio da utilização de ferramentas e linguagens de programação para o desenvolvimento de aplicações móveis.

Objetivos Gerais

  • Construir conhecimento sobre os principais conceitos do POO, através de apresentação, análises e implementações de soluções utilizando este paradigma;
  • Construir conhecimento sobre os diversos tipos de Estrutura de Dados (conceitos e algoritmos), através de análises e implementações de soluções, utilizando estruturas simples e complexas;
  • Construir conhecimento sobre os principais conceitos relacionados ao desenvolvimento de aplicações móveis, utilizando conceitos do POO e Estrutura de Dados para o desenvolvimento de aplicativos móveis.

Objetivos Específicos

  • Conhecer os principais conceitos sobre POO;
  • Analisar a aplicabilidade de POO para solução de problemas;
  • Aplicar os conhecimentos sobre POO, em aplicações reais;
  • Conhecer os conceitos e os algoritmos de Estruturas de Dados;
  • Analisar a aplicabilidade de Estrutura de Dados para solução de problemas;
  • Construir diversos tipos de Estruturas de Dados (Pilha, Fila, Lista e Árvore);
  • Compreender os conceitos relacionados ao desenvolvimento de aplicativos móveis;
  • Desenvolver aplicativos móveis utilizando conceitos de Estrutura de Dados e POO.
