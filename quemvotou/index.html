Projeto Quem Votou?

O projeto Quem Votou? visa compartilhar informações sobre os votos dos políticos brasileiros, de modo a apoiar eleições futuras como base nas ações destes enquanto representantes do povo.

O projeto se encontra em fase incial, nesta fase será possivel consultar algumas votações dos políticos na Câmara dos Deputados e no Senado Federal, além de realizar filtros por Partidos e por Estados (UF).

Esse projeto se matém através da colaboração, assim é indispensável o envio de pedidos de cadastros e compartilhamento de listas de votação para que o projeto cresça.

Atualmente esse projeto se encontra hospedado na plataforma github por coveniência e usa apenas scripts para o gerenciamento das informações. Eventualmente poderemos migrar para um servidor.

Contamos com o apoio de vocês!
